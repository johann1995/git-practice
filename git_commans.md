Describe que hacen los siguientes comandos de git, escribe su descripcion al frente en una sola linea.

git status: Este comando se usa para obtener el estado actual del repositorio

git clone: se utiliza para clonar un repositorio de Git desde un servidor remoto a tu máquina local

git pull: Git conecta con el repositorio remoto asociado y recupera las últimas actualizaciones.

git checkout: Para cambiar de una rama a otra, puedes usar "git checkout" seguido del nombre de la rama 

git log:Este comando mostrará información detallada sobre cada commit.

git branch: se utiliza en Git para listar, crear y administrar ramas en un repositorio.

git add:  se utiliza en Git para agregar cambios de archivos o directorios al área de preparación.

git commit: para confirmar o registrar los cambios que has agregado al área de preparación.

git push: para enviar los cambios confirmados (commits) desde tu repositorio local al repositorio remoto.

git merge: se utiliza en Git para combinar los cambios de una rama en otra.
